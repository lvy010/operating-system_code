# operating-system_code

操作系统相关学习与实验的聚合仓库，包含读书笔记、示例代码与内核学习记录。

## 目录
- `csapp_os_code/`：OSTEP 读书笔记与配套示例代码（虚拟化、并发、持久化）。
- `linux-core/`：从迁移 <https://github.com/lvy010/linux-core>，未来原仓库可能用于整理其他代码。
- `linux-study/`：存放Linux 内核/系统学习gitee repo 上的迁移代码

## 使用方式
- 各子目录均为独立项目，进入对应目录查看 README 即可开始。
- 代码主要基于 C/C++ 与常见构建工具（如 CMake/Make）；请按子项目说明准备依赖。

## 贡献
- 欢迎 issue / PR 交流想法或补充资料。
- 具体许可证以子目录内说明为准
